# Golden Samples

This directory contains a small set of golden samples for testing language
identification. The purpose of these files is to provide a stable,
version-controlled correctness gate for the language detection model.

## Attribution

The audio files in this directory are derived from the following sources:

- **English (`en_*.wav`)**:
  - Source: [LibriSpeech ASR corpus](https://www.openslr.org/12)
  - License: [Creative Commons Attribution 4.0](https://creativecommons.org/licenses/by/4.0/)

- **French (`fr_*.wav`)**:
  - Source: [MLS (Multilingual LibriSpeech)](https://www.openslr.org/94/)
  - License: [Creative Commons Attribution 4.0](https://creativecommons.org/licenses/by/4.0/)

- **Spanish (`es_*.wav`)**:
  - Source: [MLS (Multilingual LibriSpeech)](https://www.openslr.org/94/)
  - License: [Creative Commons Attribution 4.0](https://creativecommons.org/licenses/by/4.0/)

All files have been clipped to be under 10 seconds and converted to
16-bit PCM WAV format for consistency.
